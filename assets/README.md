# Assets

Esta pasta contém os assets do aplicativo.

## Estrutura

- `images/`: Imagens do terço (contas, cruz, etc.)
- `data/`: Arquivos JSON com configurações do terço

## Imagens Necessárias

Para um funcionamento completo, adicione as seguintes imagens em `images/`:

- `cruz.png`: Imagem da cruz
- `conta_grande.png`: Imagem de uma conta grande
- `conta_pequena.png`: Imagem de uma conta pequena
- `fundo.png`: Imagem de fundo (opcional)

## Arquivos de Dados

Os arquivos JSON em `data/` seguem o formato especificado no README principal.
